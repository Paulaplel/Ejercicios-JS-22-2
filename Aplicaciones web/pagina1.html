Ejercicio 1

En este ejercicio se muestran alertas en donde se declaran primero los valores y posteriormente se declaran las alertas

var valor_1 = 10; //se declara valor 1
var valor_2 = 5; //se declara valor 2
alert(valor_1*valor_2); //se declara la alerta
var t=valor_1*valor_2; //se declara la variable, en este caso se asigna la variable t
alert('Fin del proceso que dio como resultado' + t); //se declara la alerta
